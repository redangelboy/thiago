# thiago

Proyectos personales. Sitio en GitHub Pages: **https://redangelboy.github.io/thiago/**

## Estructura

| Carpeta    | Contenido |
|------------|-----------|
| `Staar5th/` | Entrenador de matemáticas STAAR (Grado 5) — `index.html` |

La raíz tiene un `index.html` que redirige a `Staar5th/` para que la URL principal funcione sin escribir la subcarpeta.

## Añadir otro proyecto

1. Crea una carpeta nueva (ej. `OtroProyecto/`).
2. Pon ahí tu `index.html` o archivos.
3. Enlázalo desde la página de inicio si quieres un menú (podemos cambiar el `index.html` de la raíz de “solo redirección” a una página índice con enlaces).
